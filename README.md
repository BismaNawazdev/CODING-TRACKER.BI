coding Tracker is a responsive web application built with HTML, CSS, and JavaScript
that helps users organize and track their coding journey. It allows users to add 
programming languages, learning resources, and integrations, monitor progress through
analytics, and manage their learning in a simple, user-friendly interface.
The project focuses on clean UI, responsive design, and efficient progress
tracking for developers.
