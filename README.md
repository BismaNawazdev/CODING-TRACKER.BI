📊 Coding Tracker Pro

A modern and responsive Coding Tracker built with HTML, CSS, and JavaScript. It helps developers log coding sessions, track study hours, monitor learning progress, and visualize analytics through an interactive dashboard. The project features a clean glassmorphism design, dark developer theme, and stores data locally using the browser's Local Storage.

🚀 Features
📅 Add and manage coding sessions
💻 Track programming languages and topics
⏱️ Monitor total coding hours
📊 Interactive analytics with Chart.js
📈 Progress bar for coding goals
🌙 Dark/Light mode toggle
💾 Local Storage support (no database required)
🗑️ Delete coding sessions
📱 Fully responsive design
🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6)
Chart.js
Local Storage API
